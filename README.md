# 会津若松市周辺における川霧・気象現象予測および可視化システム

本プログラムは、気象庁の過去の観測データ（Excel形式）を自動的に解析・可視化し、さらに機械学習モデルを用いて今後16日間の地点別気象現象（川霧を含む11クラス）を個別予測・出力するPythonスクリプトです。

---

## 💡 主な機能

### 1. 過去データの月別自動可視化
長期間のデータであっても視認性を損なわないよう、「年-月」ごとにデータを自動分割し、タイムラインを完全同期した2段構成のグラフを出力します。
* **① 気温・湿度・露点 × 現象コード**: 気温と露点温度の推移、および相対湿度（右軸）をプロットし、下段に各地点の現象発生状況をレーン形式で表示します。
* **② 風速・降水量 × 現象コード**: 風速の推移と降水量（右軸棒グラフ）をプロットし、下段に同様の現象レーンを表示します。

### 2. 機械学習（RandomForest）による地点別多クラス予測
* 霧の有無を判定する二値分類ではなく、元データの粒度を維持した「現象なし（/）」および「現象コード1〜10」の計11クラスの多クラス分類モデルを、各地点ごとに独立して学習します。
* RandomForestClassifierに対し、ツリー数の拡張（500本）や、データ不均衡を是正するクラス重み調整（`class_weight='balanced_subsample'`）を施した高度なパイプラインを採用しています。

### 3. 気象予報API連携による未来16日間のピンポイント予測
* 予測対象地域の緯度・経度に基づき、Open-Meteo API（無料・キー不要）から今後16日間の時間別気象予報を自動で取得します。
* 直近5日間の実測データから連続する形で、未来16日間の地点別予測グラフを出力します。

## 📊 現象コードおよび配色定義

下段の現象レーン、および予測グラフでは以下の定義に基づいてカラーマーキングが行われます。

| 現象コード | 意味 | グラフ上の表示色 |
| :---: | :--- | :--- |
| **/** | 現象なし | 薄い黄色 (`#f9e79f`) |
| **1** | 薄い川霧 | 薄い水色 (`#aed6f1`) |
| **2** | 川霧 | 水色 (`#3498db`) |
| **3** | 濃い川霧 | 濃い水色 (`#1a5276`) |
| **4** | 薄い全体霧 | 薄い灰色 (`#dcdde1`) |
| **5** | 全体霧 | 灰色 (`#909497`) |
| **6** | 全体濃い霧 | 紺色 (`#2c3e50`) |
| **7** | 薄い層雲 | オレンジ色 (`#f8c471`) |
| **8** | 濃い層雲 | 赤色 (`#d35400`) |
| **9** | 霧雨 | 薄い緑色 (`#a9dfbf`) |
| **10** | 雨 | 緑色 (`#196f3d`) |
